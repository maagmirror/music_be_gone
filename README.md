# Reggaeton Be Gone
sends packets to disable BT speakers (hopefully)

# Parts 
Raspberry Pi 3 \

DFRobot Oled 128x32 \

Push button \

BT Audio Receiver 5.0 (to test with your own BT) \

Jumper cables

# Connections
Oled SDA ->  Rpi GPIO 2
Oled SCL -> Rpi GPIO 3
Oled VCC -> Rpi VCC
Oled GND -> Rpi GND

Button pin 1 -> GPIO26
Button pin 2 -> GND

Power supply: 5V 3A
